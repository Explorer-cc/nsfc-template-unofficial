### NSFC 申请书 LaTeX 模板（非官方）

国家自然科学基金（NSFC）面上项目申请书正文模板（2026版）

搓了一个小模板... 相比[fylimas/nsfc: nsfc - 国家自然科学基金项目LaTeX模版(面青地)](https://github.com/fylimas/nsfc)使用`\ctexset`的较现代接口实现了相应的样式控制

### 项目简介

本模板使用 LaTeX 编写，符合国家自然科学基金申请书的格式要求，包括：

- 立项依据
- 研究内容
- 研究基础（含研究基础与可行性分析、工作条件、正在承担的科研项目、完成基金项目情况）
- 其他需要说明的情况

### 编译方式

推荐使用 `XeLaTeX` 编译：

```bash
xelatex mwe.tex
```

或者使用 `latexmk` 自动编译：

```bash
latexmk -xelatex mwe.tex
```

### 依赖要求

- TeX Live
- ctex 宏包（中文支持）
- 使用 XeLaTeX 编译器

### 字体要求

模板使用以下中文字体（我暂时只希望在`windows`系统上提供原生支持，原始的`.docx`模板文件中的中文字体均为「中易字体」，在`macOS`上不容易实现，**我不希望上传可能带来版权问题的字体**）：

- 宋体 (`simsun.ttc`)
- 黑体 (`simhei.ttf`)
- 楷体 (`simkai.ttf`)

### 文件说明

| 文件         | 说明                |
| ------------ | ------------------- |
| `mwe.tex`  | 主模板文件          |
| `nsfc.sty` | NSFC 样式文件       |
| `mwe.pdf`  | 编译生成的 PDF 示例 |

### 注意事项

- 申请书正文原则上不超过 30 页
- 请勿删除或改动提纲标题及括号中的文字
- 模板使用 `zhlipsum` 生成占位文本，实际使用时请替换为真实内容
